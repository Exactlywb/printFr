# printFr

My personal printf on asm
===

Ded32 - 2 sem (asm)

In this project I implement my own **printf** function (printFr)

My function supports formats

| Symbol  | Note                |
|:-------:|:-------------------:|
|%c       |print character      |
|%d       |print decimal        |
|%b       |print binary num     |
|%o       |print octal num      |
|%x       |print hex num        |
